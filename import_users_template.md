# File Excel Import Users Template

Tạo file Excel với 2 cột:
- Column A: username (alphanumeric)
- Column B: email (valid email format)

Ví dụ:
| username | email |
|----------|-------|
| john_doe | john@example.com |
| jane_smith | jane@example.com |
| alex_test | alex@test.com |

Lưu ý: 
- Hàng 1 là header (tên cột)
- Dữ liệu bắt đầu từ hàng 2
- Username và email không được trùng lặp
- Email phải hợp lệ
